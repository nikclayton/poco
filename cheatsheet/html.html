Elements and attributes

Elements

HTML pages consist of elements, nested inside one another.

Each element has a starting tag, content, and an ending tag.

<start-tag>content</end-tag>

The tags contain the name of the element.

Useful elements include:

  • h1 ... h6 - headings
  • p - paragraphs
  • strong - a type of emphasis
  • em - another type of emphasis
  • ol - ordered lists (li listitems)
  • ul - unordered lists (li listitems)

Structure

An HTML page's general structure is:

<!DOCTYPE html>
<html>
  <head>
    <title>Document title</title>
  </head>
  <body>
    ... content elements go here ...
  </body>
</html>
